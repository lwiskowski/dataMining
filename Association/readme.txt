Implementation of and Association Algorithm using the Priori Principle

Files:

attributes.java: object that contains attributes,
and what values they could hold.

Generator.java:
file to run the program. has all the methods to calculate support,
generate rules, generate itemsets, and merge them

Itemset:
object to represent an Itemset, stores the in an arraylist

Rule.java:
object for a rule. has antecedent, consequent and string representation.

enter command lines args in form, attributes.txt file.txt support, and confidence
example:
lens-attr.txt lens.txt .4 .75

to run, compile Generator.java using javac and use command line args.
